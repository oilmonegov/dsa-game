# Lessons Learned

This directory contains thematically organized lessons learned during the development of the DSA Interactive Learning Game.

## Structure

Lessons are organized by theme/category:

```
lessons/
├── README.md                    # This file
├── 01-project-setup/           # Initial project configuration
├── 02-git-and-github/          # Version control and GitHub workflows
├── 03-react-patterns/          # React component patterns and state management
├── 04-typescript/              # TypeScript tips and gotchas
├── 05-testing/                 # Testing strategies and tools
├── 06-database/                # sql.js and browser-based persistence
├── 07-visualization/           # Cytoscape.js and data visualization
├── 08-performance/             # Performance optimization
└── 09-debugging/               # Debugging techniques and tools
```

## Lesson Format

Each lesson file follows this structure:

```markdown
# [Lesson Title]

## Context
What were we trying to do?

## Problem
What issue did we encounter?

## Solution
How did we solve it?

## Key Takeaways
- Bullet points of lessons learned

## References
- Links to documentation, articles, etc.
```
